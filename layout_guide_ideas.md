text : we click on a (general) edit button, then we can write in the box and a save button saves all of them. the edit button turns all the texts of the page/panel into editing mode.

for lists (in json), there is a delete/edit buttons for each element, as well as buttons to reorder the list. we will be able to edit already existing elements, delete existing elements, reorder the list (by moving them up or down), and add new elements (with a generic empty "new element" field ready to be edited).

for values that can only be in a selection (like gender, bolleans, a list of choices) then the edit mode turns it into a dropdown, with save button.

for colours, it opens a colour palet where we can choose the new colour. (same if we have to add a new colour.)

if it is a more complex card or item, like a card containing a clour, a list of things etc. the edit button opens a modal/panel (not full screen) which contains all the editable elements of the card in a simpler interface. this modal will have a delete button, and save button.
